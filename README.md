# IAGI-Android-Mobile-Dev

A comprehensive collection of Android mobile application projects developed during the IAGI Mobile Development course.

##  Features

*   **Main Hub Activity**: Centralized navigation to all course exercises.
*   **Activity 5-1**: Implementation of `ListView`, `GridView`, and `Spinner` for country lists.
*   **Advanced UI**: Modern `RecyclerView` implementation with `CardView`.
*   **CRUD Operations**: Full Create, Read, Update, and Delete functionality using `AlertDialog`.
*   **Menus & Navigation**:
    *   **Options Menu**: Accessible from the Toolbar.
    *   **Context Menu**: Accessible via long-press.
    *   Navigation to **About** and **Contact** activities.
*   **Class Exercises**: Collection of various activities (App_Act2 to App_Act8) covering core Android concepts.

## Tech Stack

*   **Language**: Java
*   **UI Components**: Material Design, ConstraintLayout, CardView, RecyclerView.
*   **Architecture**: Activity-based navigation with Intents.

##  How to Run

1. Clone the repository.
2. Open the project in **Android Studio**.
3. Build and run on an emulator or physical device.
